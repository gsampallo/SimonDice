Version de juego Simon Dice en Arduino
*****

Los componentes utilizados son:

- Arduino Nano
- 4 led
- 4 resistencias 1k para los led
- 4 resistencias 10k
- 1 resistencia de 1k para el input
- 1 speaker
- cables
- protoboard
- push button
